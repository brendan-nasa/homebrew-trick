# Brendan-nasa Trick

## How do I install these formulae?

`brew install brendan-nasa/trick/<formula>`

Or `brew tap brendan-nasa/trick` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "brendan-nasa/trick"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
